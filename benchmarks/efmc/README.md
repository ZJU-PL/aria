# EFMC Benchmarks

This directory groups benchmark families primarily used for program
verification and invariant synthesis in `aria/efmc`.

- `Boogie/` - Boogie verification programs
- `BV/` - Bit-vector verification benchmarks
- `C/` - C verification programs
- `chc/` - Constrained Horn Clause benchmarks
- `INT/` - Integer verification benchmarks
- `KSafety/` - Relational and k-safety benchmarks
- `sygus-inv/` - SyGuS invariant-synthesis benchmarks
